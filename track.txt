//---How vagrant file works -->

config.vm.box - Operating System
config.vm.provider - virtualbox
config.vm.network - How you host sees your box
config.vm.synced_folder - How you access files from you computer
config.vm.provision - what we commands to run in vm

//----- Vagrant Network Configurations

1 > Port forwarding 
    ex : config.vm.network 'forwarded_port', guest:80, host:8080;
    maps port 80 of the VM to port 8080 on the host machine

2 > Private network (host-only network)
3 > Public network (bridged network)
